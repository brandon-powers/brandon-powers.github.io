About

Hi, I'm Brandon Powers. I'm an aspiring software engineer from right outside of Boston, Massachusetts.

I'm currently working on my Bachelors of Science at Stonehill College, majoring in Computer Science. The awesome part of programming is three-fold: first, it's a creative process, where with a simple set of tools (compiler, editor, etc.) and an idea, you can create (almost) anything. Second, while building or working on something, you inevitably come across something you don't know how to do. Figuring out the "how to" of doing something, that act of learning, is rewarding as hell. Lastly, once the creative process is finished, the learning has been done, you have the chance to have built something that is useful to people, something that can make other peoples lives that much easier

Want to work together? Talk to me!
