Py-shell

Python

Py-shell is a bash shell emulator written in Python. It only implements a subset of the shell commands -- it's not a full blown emulator. The goal of py-shell was to demonstrate the scripting capabilities of the language and its standard library. It used modules like shutil and os which were extremely useful in implementing these commands.

Another aspect of this project was to write an in-depth research paper on the Python programming language. It covers a breadth of topics about the implementation of the language and its use. It's about 30 pages long, so feel free to skip sections. The paper is up on GitHub along with the source code below.

simple-ftp

C++, gdb, valgrind

I call this project 'simple-ftp' because it's an implementation of a simple file transfer protocol, supporting commands like get, put, and delete. First, I created the protocol used for the transfer of files and commands in a text file, then implemented them in C++.

This project was important for me because it allowed me to take it through the many stages of software development. It went from a problem specification, to a design of a solution and protocol, to the implementation of that solution. It made me think about tradeoffs of doing one thing over the other, and the effect it would have on the final project. I also had to use the utilities gdb and valgrind; gdb for debugging + segmentatition faults and valgrind for memory leaks.

On a final note, if you're interested in using this application, the client is compatible on both a Linux and OS X system, while the server is only compatible on a Linux system.

brandon-powers.github.io

HTML, CSS, Jekyll, GitHub Pages

This website was created with the goal of being as simple as possible. For that reason, I used HTML and CSS to write the markup and Jekyll to build the site. I found Jekyll to be such a useful tool in that it gives you clean URL's, includes, Liquid template processing, etc. all with the performance benefits of static content with no backend. Lastly, I'm hosting this website on GitHub Pages with enforced https, which is really awesome.
